# 为什么要有团队代码规范？

虽然这些细节是小事，不会有体验或者性能上的优化，但是却体现了一个 coder 和团队的专业程度。 同时统一的代码规范，有利于后期的维护，增加阅读他人代码的便利性等。

所以不管团队有多少人，代码风格都应该师出同门！
